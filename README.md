# README

A notes for Server/Back-end Sofrware Developer

## 本笔记的初衷是建立完善的知识体系，其目的为了对应面试，整个分支中包含：

## 1. 计算机基石（数据结构、计操作系统、计算机网络、数据库）

## 2. 高级语言（Java/Python/C/C++/JS）

## 3. 软件设计（软件体系结构、设计模式）

## 4. 框架（Django、Flask、Spring...）

## 5. 人工智能（ML、RL、DL）

## 6. 大数据

## 7. 安全

## 8. 云计算（SRE）
